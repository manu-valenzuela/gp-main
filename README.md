# gp-main
**GP: GESTIÓN DE PERSONAL**

**GP** es el backend de un programa desarrollado con microservicios que gestiona el personal de una empresa contratista en construcción.

**MICROSERVICIOS DEL SISTEMA**

Este es el repositorio principal del sistema. Los repositorios que formal parte del sistema son:


1   - [gp-trabajadores](https://github.com/manu-valenzuela/gp-trabajadores)
2   - [gp-contratos](https://github.com/manu-valenzuela/gp-contratos)
3   - [gp-faenas](https://github.com/manu-valenzuela/gp-faenas)
4   - [gp-remuneraciones](https://github.com/manu-valenzuela/gp-remuneraciones)
5   - [gp-despidos](https://github.com/manu-valenzuela/gp-despidos)
6   - [gp-asistencia](https://github.com/manu-valenzuela/gp-asistencia)

9   - [gp-consulta-trabajador](https://github.com/manu-valenzuela/gp-consulta-trabajador)
